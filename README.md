# 物业管理App后台

### 内置功能

1. 用户管理：用户是系统操作者，该功能主要完成系统用户配置。
2. 角色管理：接口权限分配，设置匿名用户。
3. 登录日志：系统登录日志记录查询包含登录ip和地址。
